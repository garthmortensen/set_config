CondaManager module¶

class conda_manager.CondaManager[source]¶

Class to manage Conda environments.

Provides methods to find where the directory which stores conda environments is located on filesystem, determien if an input conda env exists by looking therein, and finally creates it under that name if it doesn’t exist.

This is meant to run in an automatated fashion without command lines, which would merely be replicating existing conda env functionality. No, instead, just provide a variable name, edit script as required, and call it.
